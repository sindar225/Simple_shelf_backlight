### Components I used in this projects:

* 3pin IR Infrared Module 
* CR2032 Copper Led Fairy Lights 2M 20 LEDs https://www.aliexpress.com/item/32831287135.html?spm=a2g0s.9042311.0.0.27424c4diCvnCd
* 9v battery
